# Random Six Fingers ⚽

Generatore di squadre per calcetto amatoriale con vincoli di cluster.

## Funzionalità
- Inserimento di 10 giocatori
- Creazione di cluster di vincolo (giocatori che devono stare in squadre diverse)
- Generazione casuale delle squadre rispettando i vincoli
- Tema chiaro/scuro
- Copia dei risultati negli appunti
- Nomi delle squadre personalizzabili

## Sviluppo Locale
Il progetto è una Single Page Application statica senza dipendenze.
Basta aprire il file `index.html` nel browser.

## Deploy
Il progetto è configurato per il deploy su Vercel.
URL: [random-six-fingers.vercel.app](https://random-six-fingers.vercel.app)

## Stack Tecnico

- CSS3 (Vanilla)
